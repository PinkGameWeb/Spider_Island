Inspiration

This game has primarily been inspired by Ark Survival Evolved as it is a game that is set on an island that houses various dangerous creatures mainly being dinosaurs. The island theme is mainly inspired by the game Animal Crossing New Horizons as that game contains a natural island environment with resources that can be collected such as fruit and resources that can be crafted into tools.

The shape of the island is a spider which is inspired by the insects of the same name. I made this design choice due to how I like spiders and it would make for an interesting shape rather than it just being a normal looking rounded island.
